Azucena. A game that runs on a 2D game engine I made.

Azucena is a game about a llama mum that runs on a game engine I made in C++. Here's the source code. It uses the SFML and Box2D libraries.

This was a university coursework for which I had to prepare a game from the ground up: pitch presentation, game design document and – of course – code. The focus of this project was on the techical side, expecially the game engine.

Like most engines, this one is built around scenes, entities and components.
